merge-pass is a package that lets you easily create post-processing effects without a lot of graphics programming knowledge. it is kind of like a preprocessor for glsl, but it is also more than that. you specify a list or tree effects and merge-pass will create a single shader or a tree of shaders. here's the repo and here's the package on npm
